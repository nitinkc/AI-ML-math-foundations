# Notebook Labs

Notebook folder: `notebooks/math-foundations/`

## Sprint 1 labs (implemented)

- `00_warmup.ipynb` - quick Python/math warmup used by later labs
- `01_tokenization.ipynb` - token-budget simulation with IT ticket examples
- `02_probability.ipynb` - score normalization and expected-value workload estimate
- `03_logits_softmax.ipynb` - logits to probabilities with temperature effects
- `04_entropy.ipynb` - uncertainty scoring and entropy-aware routing policy

## Sprint 2 labs (implemented)

- `05_variance_stddev.ipynb` - run-to-run spread analysis for prompt stability
- `06_determinism.ipynb` - argmax vs sampling with seeded reproducibility checks
- `07_regression.ipynb` - simple linear forecasting and residual-based error review
- `08_classification_calibration.ipynb` - threshold tuning with reliability checks

## Sprint 3 labs (implemented)

- `09_correlation_causation.ipynb`
- `10_sampling_controls.ipynb`
- `11_guardrails_thresholds.ipynb`
- `12_eval_in_production.ipynb`

## How to run

Open each notebook with Jupyter and run cells top-to-bottom.
Core calculations use Python standard library; visualization cells use `matplotlib`.
