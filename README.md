# AI Math Foundations

Standalone MkDocs project for the full AI math refresher track.

## Quick start

```bash
cd /path/to/math-foundations-site
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python3 -m mkdocs serve
```

> **Important:** always activate `.venv` before running `mkdocs` commands.
> Running `mkdocs` outside the venv will produce `Unrecognised theme name: 'material'`.

## Build

```bash
source .venv/bin/activate
python3 -m mkdocs build
```
